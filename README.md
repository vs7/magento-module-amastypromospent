# Add-on for Amasty_Promo

***

## Summary:

Magento1 extension. Mod for Amasty_Promo: show banner for amasty_spent salesrule condition.

***

## Installation

Log in to the Magento server, go to your Magento install dir and run these commands:
```
composer require vs7/magento-module-amastypromospent
```
